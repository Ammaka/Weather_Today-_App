# Weather Today App(FE)

A simple weather app built with Flutter that connects to a Java backend to display real-time weather data.

## Description

This app allows users to get the current weather for any location. It features a simple two-page design:
1.  **Search Page:** Enter a city to search for.
2.  **Weather Details Page:** Displays the weather information received from the backend.

## Features

*   **Real-time Weather:** Fetches and displays current weather conditions.
*   **Simple Search:** Easily look up weather for any city.
*   **Clean UI:** A straightforward two-page navigation flow.
*   **Unit Selection:** Allows users to choose their preferred measurement units (like Celsius or Fahrenheit).
