# Weather Today App 

Weather Today is a full-stack application with:
- **Frontend:** Flutter (Web)
- **Backend:** Java 
- **Containerized with:** Docker & Docker Compose

The app allows users to view rela time weather information of particular cities via a web interface

---

## Need To Have
- **Docker**
- **Git**

> ps: You do NOT need Flutter or Java installed locally to run the app with Docker.

---

how to run
- clone the repo
- cd to the repo
- edit the dockercompose.yml (by entering your API_KEY)
- ensure your docker engine is running
- docker compose up --build
