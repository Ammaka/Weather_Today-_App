Weather Today is a full-stack application with:
- **Frontend:** Flutter (Web)
- **Backend:** Java 
- **Containerized with:** Docker & Docker Compose

The app allows users to view weather information via a web interface, while the backend handles API requests.

---

## Prerequisites

have the following installed:

- **Docker**
- **Git**

> You do NOT need Flutter or Java installed locally to run the app with Docker.

---

how to run
- clone the repo
- cd to the repo
- edit the dockercompose.yml (by entering your API_KEY)
- docker compose up --build
